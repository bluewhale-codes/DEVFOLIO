# 🚀 Devfolio (Portfolio Builder Platform)

Devfolio is a modern portfolio-building platform that allows users to create, customize, and manage their personal portfolios visually — without writing code.

⚠️ **Note:** This project is currently in the **construction phase** and not fully production-ready.

---

## ✨ Features

### 🎨 1. Customizable Portfolio (Core Feature)

* Edit text directly on screen (inline editing)
* Change:

  * Font size
  * Font weight
  * Font family
  * Colors
* Real-time preview of changes
* Dynamic content rendering using React state

---

### 🎭 2. Theme Selection System

* Multiple pre-built portfolio themes
* Users can:

  * Select any theme
  * Customize it without breaking layout
* Scalable architecture for adding new themes easily

## 🛠️ Tech Stack

* **Frontend**

  * React.js
  * Tailwind CSS
  * Vite

* **Backend (Planned / Partial)**

  * Node.js
  * Express.js
  * MongoDB (Mongoose)

---

## ⚙️ Core Concepts Used

* Dynamic component rendering
* Editable UI using `contentEditable`
* State-driven styling system
* Component-based theme architecture
* Schema-driven UI (planned)

---

## 🧠 Vision

The goal of Devfolio is to build a **no-code portfolio builder** similar to tools like:

* Webflow
* Framer
* Wix (but developer-focused)
